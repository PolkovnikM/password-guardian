# Password Guardian
Простой и эффективный генератор паролей с проверкой надежности. Веб-приложение для создания безопасных паролей и оценки существующих.

Группа: БИВТ-24-4 

## Описание проекта
Password Guardian - это полнофункциональное веб-приложение, разработанное для решения двух основных задач:

### Генерация паролей
- Создание случайных паролей с настраиваемыми параметрами
- Выбор типов символов: строчные/заглавные буквы, цифры, специальные символы
- Настройка длины пароля (8-32 символа)
- Автоматическая оценка надежности

### Проверка паролей
- Анализ существующих паролей на безопасность
- Проверка по 6 критериям надежности
- Детальные рекомендации по улучшению
- Визуальная индикация уровня защиты

## Быстрый старт
### Предварительные требования
- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Веб-браузер с поддержкой JavaScript
